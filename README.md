# NJ Beach Water Quality Analysis (2021–2026)

## Overview
This project analyzes Enterococcus bacteria levels at New Jersey beaches using publicly available NJDEP data.

## Goals
- Examine trends in bacteria levels over time
- Identify seasonal patterns
- Determine how often beaches exceed safety thresholds

## Tools Used
- R (RStudio)
- Data from NJDEP beach monitoring program

## Key Questions
- When are bacteria levels highest?
- How often are beaches unsafe?
- Do trends change across years?
